### 시작하기(현재 v3)

### v2와 v3 차이

1. Vue 버전 2에는 제한된 typescript 지원, 성능 병목 현상, 까다로운 유지 보수, 제한된 스케일링 성능 등의 단점이 있습니다.
2. Vue.js v3 버전은 Vue.js v2 버전과 호환성이 없기 때문에, Vue.js v2 코드를 Vue.js v3 버전에서 사용하면 에러가 발생합니다.
